## 📥 Installation

### One-time setup (2 minutes):

1. **Download** the latest `python-tutor-extension.zip` from [Releases](https://github.com/qsert123/Python-extension-while-learning-from-tutorials/releases)

2. **Extract** the ZIP file to a permanent folder on your computer

3. **Open Chrome** and go to `chrome://extensions/`

4. **Enable "Developer mode"** (toggle switch in top right corner)

5. **Click "Load unpacked"**

6. **Select the extracted folder**

7. **Done!** 🎉 The Python Tutor icon will appear in your Chrome toolbar

### Watch Tutorial → Click Icon → Start Coding!

## 🚀 How to Use

1. **Watch any Python tutorial** on YouTube, Coursera, or Udemy
2. **Click the Python Tutor icon** in Chrome toolbar
3. **Write your Python code** in the editor panel
4. **Click "Run Code"** to see the output
5. **Learn by doing** - code along with your tutorial!
